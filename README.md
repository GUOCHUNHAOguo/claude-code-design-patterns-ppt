# Claude Code 设计模式分析 PPT

一个基于 HTML/CSS/JS 的交互式幻灯片，分析 Claude Code 的核心设计模式。

## 📊 内容概览

- Claude Code 核心定位
- 代理架构模式 (Agent Architecture)
- 工具调用系统 (Tool Use Pattern)
- 权限与安全机制
- 组件化设计
- 事件驱动架构
- 总结与启示

## 🚀 使用方法

直接用浏览器打开 `index.html` 即可演示：

```bash
open index.html
# 或
python -m http.server 8000
# 然后访问 http://localhost:8000
```

## ⌨️ 快捷键

- `→` 或 `空格` - 下一页
- `←` - 上一页
- 点击按钮也可切换

## 🎨 设计特点

- 渐变背景和毛玻璃效果
- 流畅的动画过渡
- 响应式布局
- 深色主题护眼
